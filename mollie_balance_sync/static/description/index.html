Sync mollie Balance in odoo.

How to configure

Step 1 Download the module and place the module in apps folder and install the module.
Step 2 Open Accounting > Configuration > Journals and create new journal with type bank.
Step 3 Refer this link for quick setup of organization token: https://www.droggol.com/r/mollie-token
Step 4 Fill the API key and select Bank feeds to Mollie Balance Synchronization.
DONE Your bank mollie balance are synced as a bank statement lines in odoo.

How to Use it

Once you have configured, mollie balance line will be synced with your journal time to time.
